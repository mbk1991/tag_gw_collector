# tag_gw_collector
:태그 기반 위치데이터, Gateway 데이터를 수집하고 데이터 임계 이탈을 판단하는 프로그램

### 주요 기능
<ol>
<li>데이터 수집 스케줄러 : 위치 데이터 수집 API 호출</li>
<li>데이터 임계 판단 및 알람 API 호출</li>
</ol>

# Specification
<ul>
  <li>java17</li>
  <li>Springboot 3.2.3</li>
  <li>Mysql 8.0.20</li>

  <li>json-simple 1.1.1</li>
</ul>
